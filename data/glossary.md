# Glossing Conventions

This page lists the abbreviations used in the interlinear glosses. We follow the [Leipzig Glossing Rules](https://www.eva.mpg.de/lingua/resources/glossing-rules.php).

## Abbreviations

| Abbreviation | Meaning |
|--------------|---------|
| 1 | first person |
| 2 | second person |
| 3 | third person |
| SG | singular |
| PL | plural |
| DEF | definite |
| PFV | perfective |
| IPFV | imperfective |
| FUT | future |
| PST | past |
| NEG | negation |
| Q | question marker |

## IPA and Unicode

Phonetic transcriptions use the International Phonetic Alphabet (IPA). For best display, we recommend a browser with Unicode support and optionally installing a font such as [Charis SIL](https://software.sil.org/charis/).
